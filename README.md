# NetflixGPT

An AI-powered movie browsing and recommendation platform built with **JavaScript** that combines a Netflix-like UI with Gemini-based intelligent search and suggestions.

---

## Overview

NetflixGPT is a modern web application developed using **React (JavaScript)** that allows users to explore movies, watch trailers, and receive smart movie recommendations powered by **Google Gemini AI**.  
It integrates **Firebase Authentication**, **TMDB APIs**, and **Gemini** to deliver a personalized and interactive movie discovery experience.

This project demonstrates real-world frontend architecture, API integration, authentication, and AI-driven features using modern JavaScript practices.

---

## Features

- User authentication using Firebase (Login / Signup)
- Browse trending, popular, and recommended movies
- Gemini-powered natural language movie search
- Watch movie trailers directly
- Modern UI built with React and Tailwind CSS
- Global state management using Redux Toolkit
- Fast, responsive, and scalable JavaScript-based frontend

## Tech Stack

- **Language:** JavaScript (ES6+)
- **Frontend:** React, Tailwind CSS
- **State Management:** Redux Toolkit
- **Authentication:** Firebase Auth
- **APIs:** TMDB API, Google Gemini API
- **Build Tool:** Parcel
