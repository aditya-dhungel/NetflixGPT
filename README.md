# NetflixGPT

An AI-powered movie browsing and recommendation platform that combines a Netflix-like UI with Gemini-based intelligent search and suggestions.

---

## Overview

NetflixGPT is a modern web application built using **React** that allows users to explore movies, watch trailers, and receive smart movie recommendations using **Google Gemini AI**.  
It integrates **Firebase Authentication**, **TMDB APIs**, and **Gemini** to provide a personalized and interactive movie discovery experience.

This project demonstrates real-world frontend architecture, API integration, authentication, and AI-powered features.

---

## Features

- User authentication using Firebase (Login / Signup)
- Browse trending, popular, and recommended movies
- Gemini-powered natural language movie search
- Watch movie trailers directly
- Modern UI with React and Tailwind CSS
- Global state management using Redux Toolkit
- Fast and responsive UI
